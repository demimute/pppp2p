DedupStudio Windows x64 绿色版说明

1. 适用系统
- Windows 10 / 11 x64
- 不需要安装，解压后直接运行

2. 推荐下载内容
- DedupStudio-win-x64-unpacked.zip

3. 使用方式
- 解压 zip 到任意目录
- 双击 DedupStudio.exe 启动
- 首次启动如果被系统拦截，请在 Windows 安全提示中选择“仍要运行”

4. 注意事项
- 请保持 exe 与同目录下的其他文件在一起，不要单独移动 DedupStudio.exe
- 首次启动可能会稍慢，属于正常现象
- 如果系统缺少权限，请把整个目录放到桌面、下载目录或你自己的工作目录，不要放在系统受限目录

5. 常见问题
- 打不开或闪退：确认是否下载的是 x64 版本，并且完整解压了 zip
- 被杀毒软件拦截：这是未签名桌面程序常见现象，可先加入信任后再运行
- 提示文件缺失：通常是只复制了 exe，没有连同其他运行时文件一起复制
